About SkyCheck

SkyCheck is a real-time weather information application built with Python Flask. It retrieves live weather data from the OpenWeatherMap API and presents it through a clean, responsive interface.

Features

  • Current temperature, feels-like, humidity, wind speed, pressure, and visibility
  • 5-day daily forecast with high/low temperatures and precipitation probability
  • Weather condition icons and dynamic background colours
  • Celsius / Fahrenheit toggle (no page reload)
  • Timezone-aware local time display for the searched city
  • Graceful error handling for unknown cities and network issues

Data Source

Weather data is provided by OpenWeatherMap. Data is cached server-side for five minutes to reduce API calls. For safety-critical decisions, always consult your national meteorological service (e.g. UK Met Office).

Privacy

SkyCheck collects no personal data. No cookies are set, no analytics are gathered, and no data is transmitted to any third party beyond the city name you search for.

Technology Stack

  • Backend: Python 3.10+, Flask 3.x
  • Data: OpenWeatherMap Current Weather & 5-Day Forecast APIs
  • Frontend: HTML5, CSS3 (custom properties, Flexbox/Grid), Vanilla JavaScript
  • Configuration: python-dotenv for API key management
